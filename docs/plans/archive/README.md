# Archived plans

Completed feature/design plans, moved here once the work has landed. Kept for the **rationale** — the "why we did it this way" that git history alone doesn't surface.

- These are historical. Don't edit them to reflect later changes; write a new plan instead.
- Version-numbered milestone docs live in `../milestones/`, not here. This folder is for the loose feature/design plans (sim-hub, bounded worlds, mongo cutover, etc.).
- The active working plans stay one level up in `docs/plans/`.

Convention: when a plan in `docs/plans/` is implemented, `git mv` it here.
